# llm-runner

Simple recipe how run [localai](https://localai.io/) and [anythingllm](https://anythingllm.com/)


How to run:

- clone repo
- configure you setup in [docker-compose.yml](./docker-compose.yml)
- up containers

localhost:8080 - localai ui
localhost:3001 - anythingllm ui
